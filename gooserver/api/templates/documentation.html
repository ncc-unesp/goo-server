GOO API Documentation

The Goo API consists of a set of some endpoints. To perform an action using the Goo API, you need to select a calling convention, send a request to its endpoint and some arguments, and will receive a formatted response. All request formats, listed on the API index page, take a list of named parameters.

The REQUIRED parameter auth_token is used to specify your API Key.

The optional parameter format is used to specify a response format.

Warning! Use https for ALL requests. The only acceptable Goo API URL is https://goo.ncc.unesp.br/api/.
